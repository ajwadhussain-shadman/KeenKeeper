# Project Name
Keen Keeper

## Description
Keen Keeper is a smart contact management app that helps  maintain friendship relationships by tracking when you last connected with your friends,and current situation of the friendship

## Technologies Used
 React.js  
 Tailwind CS  
 DaisyUI    
 React Toastify  
 React Router  

##  Key Features
1. Friend Tracking System  
   Track how many days have passed since your last interaction with each friend.

2.  Tagging & Status Indicators  
   Categorize friends with tags like WORK, FAMILY, HOBBY, and TRAVEL, along with status indicators such as overdue, almost due, and on-track.

3. Connect with them

   call,text and video option to connect with them
